Cache Scanner

CacheScannerJunctionUtil

Now Playing + Lyrics

NowPlayingLyricsProxy

OSC Studio

可视化 OSC 合成器：硬件遥测卡片、音乐模板、Avatar 参数、Chatbox 排版预览，1 秒自动发送。

从本地 UnityFS Bundle 直接解析三维网格，OrbitControls 自由旋转、平移、缩放。

Safe Delete

默认 dry-run，自动检测 VRChat.exe 运行状态，保留 __info 与 vrc-version。

SafeDeleteProcessGuard

NTFS Migration

用 NTFS junction 把 Cache-WindowsPlayer 迁出系统盘，无需管理员权限。

MigratorJunctionUtil

Historical Radar

离线分析 output_log，按世界切换边界重建会话：玩家进出、停留时长、化身切换。

LogParserRadarHistory

Friends & Social

好友列表 + 共处图谱 + 在线预测 + 统一 Feed + 备注/收藏/标签。

FriendAnalyticsPipeline

Plugin System

沙箱 iframe IPC 插件市场，权限声明、预安装审查、Auto-Uploader 内置。

PluginRegistrySandbox

Discord Rich Presence

自动同步当前世界/好友信息到 Discord 状态，支持隐私开关。

DiscordRpcPipeline

LogTailerDatabase

Automation Rules

事件驱动自动化规则引擎：条件-动作对，触发通知/OSC/命令。

RuleEngineEvents

HMD/Link 诊断 + 一键修复 + 自动备份/恢复 SteamVR 配置。

VrDiagnosticsSteamVrConfig
