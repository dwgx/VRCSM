Features

CAPABILITIES

Cache Scanner

Scan VRChat data categories and summarize size, file counts, timestamps, and broken junction links. Full dry-run-first safe deletion.

Bundle Preview

Inspect cache entries before deleting — structured metadata view with header badges, file tree, and UnityFS format details.

Friends & Social

VRCX-parity friends list with status buckets, trust ranks, location parsing, region display, and expandable detail rows.

Cache Migration

Move large cache folders from C: to another drive using NTFS junctions — no admin rights needed. Real-time progress tracking.

VRChat Settings

Read and write all 597 VRChat PlayerPrefs keys. Semantic editors for common settings — sliders, dropdowns, toggles with proper ranges.

Log Analysis

Live-tail VRChat logs with 1-second latency. Player join/leave timeline, avatar switches, screenshot tracking — all in real time.
